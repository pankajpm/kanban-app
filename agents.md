# Agents Guide

This project is about creating a simple Kanban web application.

## Tech Stack

- Next.js with React for the web application
- shadCN for the UI components
- Node.js for running the project and related tooling

## Development Notes

Keep the application simple, focused, and easy to run. Prefer straightforward React components and established Next.js patterns when adding features.

## Coding Style

- All code must include explanatory comments describing how the code works and why specific implementation choices were made.
- Always prefer short, small functions. If a function exceeds 10–15 lines, look for opportunities to break it into several smaller functions to aid readability.
