# Product Overview

This project is a simple single page Kanban web application.

## Core Experience

The application displays a Kanban board where users can:

- Move cards between columns
- Create new cards
- Delete existing cards

The board should be pre-populated with sample cards so the application is useful immediately after launch.

## Scope

The goal is to demonstrate a simple app experience. The application does not need:

- A database
- User accounts
- Authentication

All data can live in the client application state for the current browser session.
